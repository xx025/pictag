# PicTag 

> **高效构建图片分类数据集。**

PicTag 是一款专为本地图片分类设计的高性能工具。它能帮助您轻松整理图片收藏，是深度学习从业者和数据归档人员的得力助手。

## ✨ 功能特性

- 🚀 **高性能**: 基于现代 Web 技术和 Python 构建，即使处理大型数据集也能保持流畅。
- 🏷️ **高效标注**: 专为快速图片分类和打标设计的交互界面。
- 📂 **本地优先**: 所有数据在本地处理，充分保护您的隐私。
- 📊 **数据统计**: 内置统计工具，可视化标签分布和数据集平衡情况。
- 🌓 **自适应主题**: 自动跟随系统的深色/浅色模式设置。
- 🛠️ **跨平台**: 支持 Windows 和 Linux。
- ⏩ **自动翻页**: 标注后自动切换下一张。
- ⌨️ **自由定制**: 无限制自定义类别和快捷键。
- ⚡ **极速效率**: 人工分类效率极高，单张耗时 < 0.5s (粗略估计)。
- 🗄️ **海量支持**: 对大数据集支持友好 (已在 10k+ 数据集上测试)。
- 🌐 **双语支持**: 完整的本地化与国际化支持（中英文）。

## 使用方法

### Windows
下载后，直接双击运行即可。

### Linux
1. 下载对应的二进制文件（如 pictag-x86_64 或 pictag-arm64）。
2. 赋予可执行权限：
   ```bash
   chmod +x pictag-xxx
   ```
3. 通过命令行启动：
   ```bash
   ./pictag-xxx
   ```
   （请将 pictag-xxx 替换为实际文件名）
